⚠️ THE DANGER: Accessing nested properties without checking causes TypeError: Cannot read property 'x' of undefined, instantly crashing your entire app. One missing field from an API = white screen of death in production.
✅ THE FIX:
user?.address?.city - Optional Chaining (returns undefined if missing)
user?.address?.city ?? 'Unknown' - Nullish Coalescing (fallback value)
array?.[0] - Safe array access
func?.() - Safe function call
value ?? fallback - Only for null/undefined (NOT for false/0/'')
value || fallback - For ANY falsy value (false/0/''/null/undefined)
