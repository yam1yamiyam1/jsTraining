⚠️ THE DANGER: Every event listener without cleanup is a memory leak. Adding listeners inside components without removing them causes your app to slow down, consume gigabytes of RAM, and eventually crash. This is React useEffect's #1 purpose: cleanup!
✅ THE FIX:
Delegation: Attach ONE listener to parent, check e.target
Cleanup: element.removeEventListener(event, handler)
Critical: Can't remove anonymous functions! Use named functions.
React Pattern: useEffect() return function = cleanup
e.target.matches('selector') - Check if clicked element matches
e.target.closest('selector') - Find parent matching selector
